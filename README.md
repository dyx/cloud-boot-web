# 微服务前端脚手架
## 依赖
### 环境
- Node 18+
### JS包
- TypeScript 5.7
- Vue 3.5
- Vite 6.2
- VueRouter 4.5
- ElementPlus 2.9
- Pinia 3.0
- Axios 1.8
- @antfu/eslint-config 4.5
## 项目结构
```
.
├── public
├── src
│   ├── api 接口
│   ├── assets 静态资源
│   ├── components 公共组件
│   ├── layout 布局
│   ├── router 路由
│   ├── stores 存储
│   ├── types 类型
│   ├── utils 工具相关
│   ├── views 页面
│   ├── App.vue 入口页面
│   ├── main.ts 入口文件
│   └── vite-env.d.ts
├── index.html
├── package.json
└── vite.config.ts
```
## 运行
```bash
npm install
npm run dev
```
